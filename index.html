About Me

Johnathan Mueller is a capable Game Developer that loves to make games, art, sound effects, etc.

His skill set includes: 2d Sprite Painting, image editing, 2D Vector Graphics, 3d modeling, game coding (languages include: C, C#, Lua, ActionScript2.0 / 3.0, python, gd), as well as a few other skills in windows and linux

the game engine's that I've used over the years have been: MX Flash 2004, GameMaker8 Pro, RPG Maker VX Ace, DX Studio, Visual Basic, Love2d, Unity, UE4, Cryengine, Godot, Maratis, BGE, Armory, Jmonkey, Torque3d, RenPy, Ogre, Panda3d, and a few others
